Privacy Policy for VoIQ

Last Updated: January 9, 2026

VoIQ: The Vocabulary Intelligent Quest AI system is committed to protecting your privacy.

1. Data Storage & Privacy

VoIQ is an offline-first application. All user-generated content, including vocabulary words, categories, attempt history, and scores, is stored strictly on the user's device in a local SQLite (Room) database. VoIQ does not collect, transmit, or store any personal data on external servers.

2. Third-Party API Integration

To provide advanced features, the App interacts with external APIs:

  • Google Gemini AI: Used to generate clues. Only the relevant vocabulary word is sent to Google's servers. No personal user data or full database content is shared.
  • Dictionary/Datamuse APIs: Used for "Smart Add" definitions.
  • Google Play services: Collects basic installation metrics and crash reports for app improvement purposes.

3. API Keys

If a user provides their own Google Gemini API Key, it is stored in an encrypted format on the device using Android's EncryptedSharedPreferences (Keystore) and is never shared, uploaded, or visible to the developers.

4. Data Control

Users have absolute control over their data. Deleting the app or clearing its data via Android System Settings will permanently remove all stored vocabulary and history.

5. Contact

If you have any questions regarding this privacy policy, you may contact the developer via the official app store support channel.
